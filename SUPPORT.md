# Support

Bellweather Audio Core is maintained by Bellweather Studios as a single-maintainer
open-source C++ audio library release.

## Security

Report security issues privately; see [`SECURITY.md`](SECURITY.md).

## Bugs And Questions

Use GitHub Issues for reproducible bugs, portability reports, and documentation
gaps. Include your operating system, compiler, CMake version, exact command, and
the failing output.

Support is best-effort. Issues that affect the documented library preset,
public-surface targets, metering conformance, framework-boundary proof, or the
Barometer source build take priority. This project does not provide support for
prebuilt plugin binaries, installers, signing, notarization, or marketplace
packaging because those artifacts are not part of the public v1.0.0 release.
